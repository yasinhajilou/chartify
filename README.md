# Spotify Top 200 - All Time

A React app displaying Spotify's Top 200 most streamed songs of all time.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-7-646CFF?logo=vite)

## Features

- Live data from Spotify's all-time streaming charts
- Search by song title or artist name
- Responsive design for all devices
- Clean dark theme with Spotify-green accents
- Formatted stream counts (hover for full numbers)

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19 with Hooks
- Vite
- Plain CSS
